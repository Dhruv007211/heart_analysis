Statistical Metric Dictionary

These clinical features map directly to the active components rendered inside your Tableau multi-dimensional layouts:

1. Chest Pain (CP Classification):

Categorized from Type 0 to Type 3 representing Asymptomatic, Typical Angina, Atypical Angina, and Non-anginal physiological pain models.

2. Serum Cholesterol (chol):

Measured in mg/dl. Serum baselines exceeding 200 mg/dl signify high risk factors monitored dynamically across cross-filters.

3. Max Heart Rate (thalach):

Peak metabolic performance heart rate achieved during continuous clinical stress-testing protocols.

4. Exercise Angina (exang):

Binary variable (1 = Yes; 0 = No) reflecting immediate ischemic constraints on cardiac muscles during stress exercises.
